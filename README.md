# To-Do List App

A simple Python To-Do List app that lets you add, view, and delete tasks. Tasks are saved to a file (`tasks.txt`) so your list persists between runs.

## Features
- Add tasks
- View tasks
- Delete tasks
- Tasks are stored in a file for persistence

## Requirements
- Python 3.x

## How to Run
1. Clone the repo or download the file.
2. Run the app in terminal:
```bash
python todo_file.py
